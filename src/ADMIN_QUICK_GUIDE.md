### 1. Overview

### 2. Projects
**Add New Project**:
1. Click "Add Project"
2. Fill in:
   - Title (required)
   - Description (required)
   - Thumbnail URL or upload image
   - Category (UI/UX, Branding, SEO, Framer, Development)

3. Click "Save"

**Edit/Delete**:
- Click Edit icon to modify
- Click Trash icon to delete (with confirmation)

---

### 3. Posts (Blog)
**Add New Post**:
1. Click "Add Post"
2. Fill in:
   - Title (required)
   - Excerpt (short summary)
   - Content (full article) (required)
   - Thumbnail URL or upload image
   - Tags (comma separated, e.g., "design, web, tutorial")
   - Read time (in minutes)
3. Click "Save"

**Tips**:
- Use engaging titles
- Keep excerpts under 150 characters
- Add relevant tags for better organization
- Estimate read time: ~200 words per minute

## üîß Troubleshooting

### Common Issues:

**Can't Login?**
- Check username: `aurangzebsunny` (no spaces, lowercase)
- Check password: `sunny878`
- Make sure URL has `#admin`
- Clear browser cache and try again

**Image Not Showing?**
- Verify URL is publicly accessible
- Check image URL ends with .jpg, .png, etc.
- Try uploading instead of using URL
- Check file size (keep under 5MB)

**Changes Not Appearing?**
- Click "Save" or "Save Changes" button
- Refresh the main website page
- Clear browser cache
- Check browser console for errors

**YouTube Video Not Playing?**
- Use full YouTube URL, not embed code
- Check video is not private
- Try both formats: `youtube.com/watch?v=` or `youtu.be/`

**WhatsApp Link Not Working?**
- Include country code (e.g., +1 for US)
- Remove spaces and special characters
- Format: `+1234567890`
- Don't include `https://wa.me/` (added automatically)
