## How to Add Content

### Adding a New Project
1. Go to "Projects" in the sidebar
2. Click "+ Add New Project"
3. Fill in:
   - Project title
   - Description
   - Category (Web, Mobile, Design, etc.)
   - Technologies used
   - Project URL (optional)
   - Images (upload or paste URL)
4. Click "Save"

### Adding a Blog Post
1. Navigate to "Posts"
2. Click "+ New Post"
3. Enter:
   - Post title
   - Content/body
   - Excerpt (preview text)
   - Featured image
   - Tags
   - Read time (minutes)
4. Click "Publish"

### Adding Videos
1. Go to "Videos" section
2. Click "+ Add Video"
3. Paste YouTube or Vimeo URL
4. Add title and description
5. Save to display in gallery

### Managing Q&A
1. Select "Q&A" from menu
2. Add question and answer pairs
3. Optional: Set category (General, Pricing, Services, etc.)
4. Set order/priority
5. Save changes
